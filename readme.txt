Préréquis: 
Avoir Node, Android Studio installés et une carte NFC MifareClassic

Pour lancer le serveur,

cd ./SnackSpeak

node app.js 

Récupérer l'adresse IP de l'ordinateur : Le mettre dans 

nfc.mbds.org.snacker.CurrentMode => URL

Activer Debug USB et NFC dans le portable si ce n'est pas le cas.
Lancer l'application dans Android Studio
